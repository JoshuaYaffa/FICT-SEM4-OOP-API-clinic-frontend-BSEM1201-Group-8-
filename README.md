This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## 🌍 Overview

Smart Clinic Service Frontend is a comprehensive healthcare platform that provides an intuitive interface for patients, doctors, clinic managers, and administrators to manage healthcare services efficiently. The application is specifically designed to address healthcare accessibility challenges in Sierra Leone.

## ✨ Features

### 👤 Patient Features
- ✅ Browse and search clinics by district and services
- ✅ View detailed doctor profiles with specializations
- ✅ Book, view, and cancel appointments
- ✅ Rate and review clinics
- ✅ Manage personal profile with image upload
- ✅ Change password securely
- ✅ Receive real-time notifications
- ✅ View appointment history

### 👨‍⚕️ Doctor Features
- ✅ View and manage patient appointments
- ✅ Approve, complete, or cancel appointments
- ✅ Reply to patient messages
- ✅ View patient list and history
- ✅ Access annual performance reports
- ✅ Manage professional profile with photo
- ✅ Change password

### 📋 Clinic Manager Features
- ✅ Manage clinic information and services
- ✅ Add, edit, and remove doctors
- ✅ View and manage clinic appointments
- ✅ Respond to patient reviews
- ✅ Generate and view annual reports
- ✅ Edit clinic settings and hours

### 👑 Admin Features
- ✅ Full CRUD operations for clinics
- ✅ Full CRUD operations for doctors (with email/password)
- ✅ User management and role assignment
- ✅ View system analytics and charts
- ✅ Manage all appointments
- ✅ Moderate reviews
- ✅ View audit logs
- ✅ Manage annual records
- ✅ Team member image management
